Palette Tab

This is a collection of Preset Elements, Layouts and Focus's. Drag them onto the Tree view of the Element Builder or the Layout Builder to create a copy in the current UI file of this item.

The Palette is split into several sections:

  • Layouts: These are Elements that contain a Layout.
    • Layered: The elements children are layered on top of one another in order
    • Vertical: The elements children are arranged vertically in order
    • Horizontal: The elements children are arranged horizontally in order
    • Grid: The elements children are placed into an x by y grid
    • ScrollableVertical: The elements children are arranged vertically in order in a scrollable window
    • White_Background: An element containing the standard background components.
  • Controls: These are Standard Control Elements
    • Element: A blank base element
    • Image: An element that holds a texture or a material
    • Button: An element that can be clicked on
    • Label: An element that holds a line of text
    • Progress: A progress bar
    • Slider: A slider bar
    • TabBar: A set of buttons, at most one of which is active
  • Dialogs: A dialog is a self contained window
    • alert_dialog: A dialog that asks a question.
  • ChangeLayout: These change the type of layout in an element
    • NoLayout: This changes an element to have no layout (i.e. allow free positioning of children)
    • Layered: This changes an element to have a layered layout
    • Vertical: This changes an element to have a vertical layout
    • Horizontal: This changes an element to have a horizontal layout
    • Grid: This changes an element to have a grid layout
  • ChangeFocus: These change the type of focus handler in an element
    • NoFocus: This changes an element to have no focus handler
    • FocusHandler: This changes an element to have a basic focus handler this sets focus on the first item
    • FocusHandler2D: This changes an element to have a 2D focus handler this moves the focus in 2D
    • FocusHandlerTab: This changes an element to have a Tab focus handler this moves the focus backwards and forwards
