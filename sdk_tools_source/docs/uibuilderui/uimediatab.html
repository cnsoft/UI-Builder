Media Tab

This is a collection of assets. Drag them onto the Tree view of the Element Editor to set
the respective property on that element. The types of media are:

  • texture: A list of all the textures referenced in the group hierarchy
  • gxfont: A list of all the fonts referenced in the group hierarchy
  • material: A list of all the materials referenced in the group hierarchy
  • styles: A list of all the styles referenced in the group hierarchy
