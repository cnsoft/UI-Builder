Laying-Out Controls

Layouts are used within IwUI to arrange child elements within the available space. If a control is not within a layout it can be moved and re-sized freely. However if a control is within a layout it can re-size and re-position itself automatically depending on the space available.

Applications can be made up from elements and controls nested within a layout tree so that the application will adjust suitably for a wide range of screen sizes.

Aligning Controls

With some layouts it may not always be possible to align a control to all edges of screen. Horizontal and Vertical layouts arrange the controls one after another and it is not possible to align a control with the right or bottom edge respectively. In the case of the Grid layout it is important to set the sizeToSpace property true to ensure the grid expands to fit space available.

How to align controls on a Panel

  1. Add a Panel by dragging it from the Palette onto viewer.
  2. Change the Panels layout by dragging a Layered layout from the ChangeLayout section of the Palette onto the element named "Base" in the element hierarchy.
  3. Ensure the Base element is highlighted and drag any of the Controls from the Palette onto the Viewer (it be aligned automatically)

Aligning an existing control

  1. Switch to the Element Editor
  2. Select the element or control in the Viewer or Element Hierarchy
  3. Adjust the alignV property in Element Editor
  4. Click the Refresh Viewer Data icon if the Viewer does not update

Using a Grid layout

  1. Add a Grid layout by dragging it from the Palette onto the viewer 
  2. Switch to the Layout Editor
  3. Select CIwUILayoutGrid in the Layout Hierarchy
  4. To change the number of rows or columns edit the numRows or numColumns property
  5. To ensure the grid expands to fit the entire screen set the sizeToSpace property to true
  6. To change a controls row or column select the control within Viewer, Element Hierarchy or Layout Hierarchy.
  7. Change the row or column property
  8. Change the rowSpan or columnSpan property to allow the control the span multiple rows or columns

Changing the background

  1. Add a Layered layout by dragging it from the Palette onto the Viewer or into the Element Hierarchy
  2. Add an Image by dragging it from the Palette onto Viewer into the Layered layout on the Element Hierarchy
  3. Switch to the Element Editor
  4. Select the Image in the Element Hierarchy
  5. Set the sizeToContent property to false
  6. Click the Change icon on the texture property
  7. Select -=new=-
  8. Select your .group file
  9. Choose a texture file (tga, bmp, gif or png) and press Open
  10. Click the Reset Viewer icon to update the texture
