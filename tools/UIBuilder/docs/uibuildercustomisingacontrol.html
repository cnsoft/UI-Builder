Customising a Control

Adding a texture

  1. Select the control on the viewer or within the element hierarchy
  2. Switch to the Element Editor
  3. Find the texture property
  4. If the property is greyed out, click the Override icon
  5. Click the Change icon
  6. Select -=new=-
  7. Select your .group file
  8. Choose a texture file (tga, bmp, gif or png) and press Open
  9. Click the Reset Viewer icon to load the texture

Adding a font

  1. Select the control on the viewer or within the element hierarchy
  2. Switch to the Element Editor
  3. Find the font property
  4. Select -=new=-
  5. Select your .group file
  6. Choose a gxfont file (created by the Font Builder) and press Open
  7. Click the Reset Viewer icon to load the font

Changing a Control's colour

  1. Add a Control by dragging it from the palette onto the viewer
  2. Select the Control on the viewer or within the element hierarchy
  3. Switch to the Element Editor
  4. Change the RGBA values or click the icon to bring up the colour picker

Re-sizing a button

If a button is not within a layout its minimum and maximum size must be overridden before it can be resized:

  1. Select the button on the viewer or within the element hierarchy
  2. Switch to the Element Editor
  3. Find the sizeMax property
  4. Click the Override icon
  5. Specify a larger size or (-1,-1) if you wish there to be no maximum size
  6. Repeat steps 3 - 5 for the sizeMin property
  7. If the button is not within a layout it can be resized by adjusting the size property of re-sizing the button within the viewer
  8. If the button is within an layout it will expand to it's sizeMax property if the space is available

Changing a button texture

  1. Add a button by dragging it from the palette onto the viewer
  2. Select the button on the viewer or within the element hierarchy
  3. Switch to the Element Editor
  4. Find the greyed out buttonUp property
  5. Click the override icon
  6. Click the change icon
  7. Select -=new=-
  8. Select your .ui file
  9. Enter a name (e.g '<button_texture>') and press OK
  10. Switch to the Style Editor
  11. Change the texture property
  12. Select -=new=-
  13. Select your .group file
  14. Choose a texture file (tga, bmp, gif or png) and press Open
  15. Click the Reset Viewer icon to update the texture

Adding an icon to a button

  1. Add a button by dragging it from the palette onto the viewer
  2. Select the button on the viewer or within the element hierarchy
  3. Switch to the Element Editor
  4. Set the caption to blank
  5. Drag an Image from the palette onto the button
  6. Change the texture on the Image to for example deletekey_symbol
  7. Position the Image on the button manually

In order to re-size the icon:

  1. Select the Image on the viewer or within the element hierarchy
  2. Switch to the Element Editor
  3. Set the sizeToContent to false
  4. Re-size the icon within the viewer or by changing the size property

In order to have the icon centre automatically:

  1. Switch to the Layout Editor
  2. Change the layout on the button by dragging ChangeLayout/Layered onto the button in the hierarchy
  3. Select the Image
  4. Set the alignV property to middle
  5. Set the alignH property to centre
