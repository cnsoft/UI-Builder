Material Editor Tab

This is very similar to the Element Editor except it is a list of all materials stored in any MTL files in the project. It is not possible to drag and drop with this tree.

The buttons on the tool bar are:

  • Save: Saves the file of the currently selected item
  • Reload: Abandons changes and reloads the current UI file
  • Refresh: Re-sends the current Item to the viewer
  • Add Material: Add a new, blank material
  • Check Usage: This pulls up a dialog showing which property sets are currently in use and
    which are not.
