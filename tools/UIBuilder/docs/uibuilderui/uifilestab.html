Files Tab

The Files Tab lists the files within the current Root Group. Group files are shown with their contents displayed in a hierarchy. UI files can be opened for editing by double clicking the file within the Files Tab.

Right-Click Menu

The Files Tab also includes a right-click menu. This contains the following options:

  • Add File... - Adds an existing file to the group file under the cursor, or to the root group if not over a group file.
  • New Group File... - Adds a new group file to the group file under the cursor, or to the root group if not over a group file.
  • New UI File... - Adds a new UI file to the group file under the cursor, or to the root group if not over a group file.
  • New Material File... - Add a new Material file to the group file under the cursor, or to the root group if not over a group file.
  • New Property Set File... - Add a new Property Set (ITF file) file to the group file under the cursor, or to the root group if not over a group file.
  • Remove - Remove the file under the cursor from its parent group.
  • View Source - View this file in the Source View Tab.
