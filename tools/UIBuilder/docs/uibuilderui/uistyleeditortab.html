Style Editor Tab

This is very similar to the Element Editor except it is a list of all property sets stored in any UI or ITX files in the project. It is not possible to drag and drop with this tree.

The buttons on the tool bar are:

  • Save: Saves the file of the currently selected item
  • Reload: Abandons changes and reloads the current UI file
  • Refresh: Re-sends the current Item to the viewer
  • List: List of stylesheets
  • Add PropSet: Add a new, blank property set
  • Check Usage: This pulls up a dialog showing which property sets are currently in use and
    which are not.
  • Add Stylesheet: This adds a new stylesheet to the project. First you select the location of the file that the stylesheet will be saved into, then select the group to add this file to. Then optionally select which stylesheet to parent this stylesheet from (if there are any).
